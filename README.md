# Explainable Fake News Detection

## Project Description
This project aims to build a deep learning system for fake news detection with interpretable outputs. The goal is to classify text as real or fake while providing explanation signals that help users understand the model’s decision.

## Repository Structure
- data/ : raw or sample data
- notebooks/ : Jupyter notebooks
- src/ : helper scripts and data pipeline code
- ui/ : interface code
- results/ : plots and outputs
- docs/ : diagrams and visuals

## Installation
Install dependencies with:

pip install -r requirements.txt

## How to Run
Open and run:

notebooks/setup.ipynb

## Dataset Information
The project will use public fake news detection datasets for text classification. Final dataset selection may be refined during development.
